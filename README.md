Note for the mini project:

To quickly build and run, use command in terminal:
g++ main.cpp src/possible.cpp src/grid.cpp -Iinclude && ./a.out

or, build and run a new excutable using cmake, use commands in terminal:
1. cd build
2. cmake ..
3. make
4. ./SudokuSolver
