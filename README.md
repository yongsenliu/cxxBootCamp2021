# cxxBootCamp2021
This repo is for storing all assignments from C++ Boot Camp 2021 to Github.
- Yongsen Liu

To run the demo for the mini project, run commands in terminal:
1. cd build
2. ./SudokuSolver

or, build a new excutable, run commands in terminal:
1. cd build
2. cmake ..
3. make
4. ./SudokuSolver
